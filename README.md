Identidade da Saúde

Identidade da Saúde é uma landing page responsiva criada para apresentar serviços de design visual personalizados para profissionais da área da saúde. O projeto foi prototipado no Figma, desenvolvido em HTML e CSS, e todas as imagens foram criadas manualmente usando Illustrator e Photoshop.

Sobre o Projeto

O objetivo é oferecer uma experiência visual moderna e clara, destacando a identidade da marca e incentivando o usuário a entrar em contato via WhatsApp. O layout é totalmente adaptável para desktop e mobile, garantindo boa experiência em qualquer dispositivo.

Tecnologias

HTML5 – Estrutura da página.

CSS3 – Estilização e responsividade.

Figma – Prototipagem do layout.

Adobe Illustrator & Photoshop – Criação das imagens e elementos gráficos.

Funcionalidades

Layout responsivo para desktop e mobile.

Tipografia personalizada e gradientes nos textos principais.

Botão de contato direto via WhatsApp.

Conteúdo organizado e visualmente atrativo para o usuário.

Estrutura do Projeto
/Assets
  /fonts        -> Fontes usadas no projeto
  /images       -> Imagens criadas para desktop e mobile
index.html      -> Estrutura HTML
style.css       -> Estilos CSS

Como Rodar

Clone o repositório:

git clone <URL_DO_REPOSITORIO>


Abra o arquivo index.html em qualquer navegador moderno.

Mantenha a pasta Assets no mesmo diretório para que as imagens e fontes funcionem corretamente.
